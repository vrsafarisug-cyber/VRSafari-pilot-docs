# VRSafari Africa - UNICEF Pilot Documentation

This project uses existing off-the-shelf VR education apps and requires no custom software development.

**Technology Stack:**
1. Hardware: Refurbished Meta Quest 2, Android smartphones
2. Software: Merged EDU, Labster Mobile, YouTube VR
3. Content: Pre-downloaded for 100% offline use in rural schools

**Pilot Scope:** 200 students, 4 schools in Mukono, 90 days, $5,000

**UNICEF Deliverables:**
- Technical setup guides for VR headsets
- UNEB curriculum mapping for 20 science lessons
- Impact data: test scores, attendance, payment rates

All content aligns with Uganda’s S1-S4 science syllabus. Technical documentation will be added during pilot implementation.
